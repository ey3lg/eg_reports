# Advanced Report System [FREE]

**__Features:__**
- **Auto screenshot on create report**
- **Rating & Live chat**
- **Discord logs via webhooks**
- **Secured and clean code**
- **Admin panel with statistics**
- **QuickChart integration for analytics**
- **Report archival system**
- **Day-of-week performance tracking**
- **Multi-language support (EN, ES, HE)**

**__Requirements:__**
- [oxmysql](https://github.com/overextended/oxmysql)
- [ox_lib](https://github.com/overextended/ox_lib)
- [community_bridge](https://github.com/TheOrderFivem/community_bridge)
- [screenshot-basic](https://github.com/citizenfx/screenshot-basic)

**__Installation:__**
1. Download and extract the resource to your resources folder
2. Add `ensure eg_reports` to your server.cfg
3. Import the SQL schema from `sql/schema.sql`
4. Configure webhooks in `server/config.lua`
5. Customize settings in `shared/config.lua`
6. Restart your server
